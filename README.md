# threejs

## 환경  

* Visual Studio Code 이용 (with Live Server 확장)  
* html + ES6 모듈을 이용하여 구현(번들러 사용 안 함)  


## 실행  

* npm install을 통해서 의존성 업데이트  
* 각 html을 Open with Live Server 이용 열어서 실행
